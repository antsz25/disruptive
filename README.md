#BackEnd
Comandos para ejecutar
  - npm install (si es la primera vez)
  - npm run dev
El backend utiliza un archivo .env para resguardar las variables sensible. Dicho archivo tiene las variables como:
  - PORT
  - SECRET
  - DATABASE_NAME
  - DATABASE_USER
  - DATABASE_PASSWORD
